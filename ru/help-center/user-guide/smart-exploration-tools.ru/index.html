Инструменты умного исследования (Только для мобильных устройств)

VoxTour.ai включает интеллектуальные инструменты мобильного исследования, которые помогают автоматически находить истории вокруг вас или распознавать достопримечательности с помощью ИИ.

Эти инструменты доступны в меню Smart Tools внутри мобильного приложения.

Доступ к инструментам умного исследования

  1. Откройте мобильное приложение VoxTour.ai
  2. Откройте экран карты
  3. Нажмите кнопку Smart Tools
  4. Выберите:
    • Navigator
    • Frame

Navigator

Navigator обеспечивает автоматическое повествование с учетом местоположения во время прогулки или исследования.

Как работает Navigator

Когда вы активируете Navigator:

  • Система определяет ваше текущее местоположение
  • VoxTour.ai ищет ближайшие точки интереса (POI) вокруг вас
  • Когда вы перемещаетесь и пересекаете границы POI:
    • Приложение автоматически запускает историю
    • Система определяет с какой стороны вы приближаетесь
    • История адаптируется в зависимости от вашего местоположения и движения

Navigator позволяет естественно исследовать города без ручного выбора остановок.

Как использовать Navigator

  1. Откройте Smart Exploration Tools
  2. Нажмите Navigator
  3. Разрешите доступ к местоположению при необходимости
  4. Начните движение — VoxTour.ai будет автоматически вас сопровождать

Frame (Распознавание изображений)

Frame использует распознавание изображений с помощью ИИ для определения достопримечательностей и создания аудиоисторий.

Как работает Frame

Когда вы используете Frame:

  • Вы загружаете или делаете снимок достопримечательности или объекта
  • VoxTour.ai анализирует изображение с помощью ИИ
  • Система определяет местоположение или объект
  • Связанная история или объяснение генерируется автоматически

Frame позволяет спонтанно исследовать без поиска или просмотра туров.

Как использовать Frame

  1. Откройте Smart Exploration Tools
  2. Нажмите Frame
  3. Загрузите или сделайте снимок
  4. Подождите, пока ИИ обработает запрос
  5. История будет создана после завершения распознавания

Советы для наилучших результатов

  • Включите службы геолокации при использовании Navigator
  • Используйте четкие и хорошо освещенные фотографии при использовании Frame
  • Встаньте лицом к достопримечательности для лучшей точности распознавания
  • Дайте ИИ несколько секунд для обработки при использовании Frame

⚠️ Инструменты умного исследования доступны только на мобильных устройствах.
