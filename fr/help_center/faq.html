Questions Fréquemment Posées

VoxTour est-il gratuit ?

Q : VoxTour est-il gratuit ?

R : Oui, VoxTour.ai offre une expérience gratuite qui vous permet d’explorer des villes et d’écouter de nombreuses histoires audio sans frais.

Avec la version gratuite, vous pouvez :

  • Parcourir les villes et les visites disponibles
  • Prévisualiser les points d’intérêt (POIs)
  • Écouter de nombreuses histoires audio
  • Utiliser les outils d’exploration intelligente tels que Navigator et Frame
  • Enregistrer votre progression et vos favoris après vous être connecté

Certaines visites premium ou créées par des partenaires peuvent nécessiter un achat ou un abonnement si elles sont indiquées comme contenu payant.

💡 Astuce :
Vous pouvez explorer des villes et prévisualiser des visites avant de vous connecter. Un compte est requis uniquement lorsque vous souhaitez commencer à écouter l’audio ou enregistrer votre progression.

Dois-je créer un compte pour utiliser VoxTour ?

Q : Dois-je créer un compte pour utiliser VoxTour ?

R : Non, vous n’avez pas besoin de créer un compte pour explorer VoxTour.ai. Vous pouvez parcourir les villes, consulter les visites et prévisualiser les points d’intérêt sans vous connecter.

Cependant, un compte est requis si vous souhaitez :

  • Commencer à écouter des histoires audio
  • Enregistrer votre progression
  • Ajouter des visites et des lieux à vos favoris
  • Accéder à votre contenu sur plusieurs appareils

La création d’un compte est rapide et gratuite.

💡 Astuce :
Vous pouvez explorer des villes et prévisualiser des visites avant de vous connecter. La connexion est requise uniquement lorsque vous souhaitez commencer à écouter ou enregistrer votre activité.

Puis-je utiliser VoxTour avant de voyager dans une ville ?

Q : Puis-je utiliser VoxTour avant de voyager dans une ville ?

R : Oui, vous pouvez utiliser VoxTour.ai avant d’arriver à votre destination. Vous pouvez explorer des villes, parcourir les visites disponibles et prévisualiser les points d’intérêt depuis n’importe où.

Avant de voyager, vous pouvez :

  • Rechercher et explorer des destinations
  • Prévisualiser les itinéraires et les arrêts des visites
  • Lire les descriptions des points d’intérêt
  • Choisir vos visites préférées à l’avance
  • Planifier votre itinéraire de voyage

Cependant, certaines fonctionnalités comme Navigator nécessitent que vous soyez physiquement proche des lieux pour déclencher automatiquement les histoires.

💡 Astuce :
Planifier des visites avant votre voyage vous aide à gagner du temps et à créer une meilleure expérience une fois sur place.

Comment démarrer une visite dans VoxTour ?

Q : Comment démarrer une visite dans VoxTour ?

R : Démarrer une visite dans VoxTour.ai est simple et flexible. Suivez ces étapes :

  1. Recherchez une ville ou une destination
  2. Sélectionnez une visite dans la liste disponible
  3. Ouvrez la page de la visite
  4. Appuyez ou cliquez sur Start Tour
  5. Choisissez un arrêt et appuyez sur Visit pour commencer à écouter

Vous pouvez commencer à partir de n’importe quel arrêt et explorer les visites dans n’importe quel ordre.

Pour une explication détaillée étape par étape, consultez la section Guide d’utilisation

Pourquoi la visite ne démarre-t-elle pas automatiquement ?

Q : Pourquoi la visite ne démarre-t-elle pas automatiquement ?

R : Dans VoxTour.ai, les visites sont conçues pour vous offrir une flexibilité totale. L’ouverture d’une visite ne lance pas automatiquement l’audio car vous pouvez choisir où et comment commencer l’écoute.

Pour commencer l’écoute, vous devez :

  1. Ouvrir la page de la visite
  2. Sélectionner un arrêt dans la liste
  3. Appuyer ou cliquer sur Visit
  4. Le lecteur audio s’ouvrira et l’histoire commencera

Cela vous permet de commencer à partir de n’importe quel arrêt et d’explorer les visites dans n’importe quel ordre.

💡 Astuce :
Pour une explication détaillée étape par étape, consultez la section Guide d’utilisation

Puis-je commencer une visite à partir de n’importe quel arrêt ?

Q : Puis-je commencer une visite à partir de n’importe quel arrêt ?

R : Oui, VoxTour.ai vous permet de commencer une visite à partir de n’importe quel arrêt. Les visites sont conçues pour être flexibles, vous pouvez donc écouter dans n’importe quel ordre selon votre position ou vos préférences.

Pour commencer à partir de n’importe quel arrêt :

  1. Ouvrez la page de la visite
  2. Parcourez la liste des arrêts
  3. Appuyez ou cliquez sur Visit à côté de l’arrêt souhaité
  4. L’histoire audio de cet endroit commencera

Cela facilite l’accès à une visite même si vous êtes déjà proche d’un lieu spécifique.

💡 Astuce :
Pour une explication détaillée étape par étape, consultez la section Guide d’utilisation.

Puis-je écouter VoxTour hors ligne ?

Q : Puis-je écouter VoxTour hors ligne ?

R : Oui. Vous pouvez écouter les visites VoxTour hors ligne si vous ouvrez la visite lorsque vous disposez encore d’une connexion Internet. La visite est alors mise en cache sur votre téléphone.

La dernière visite ouverte est enregistrée sur votre appareil pendant un maximum d’un mois.

Une connexion Internet minimale peut être requise pour certaines fonctionnalités, mais même une eSIM gratuite ou des données mobiles de base sont généralement suffisantes.

Comment changer la langue ?

Q : Comment changer la langue ?

R : Vous pouvez changer la langue de toute l’application VoxTour.ai dans le menu Paramètres.

Pour changer la langue de l’application :

  1. Ouvrez le menu de votre profil
  2. Allez dans Settings
  3. Sélectionnez Language
  4. Choisissez votre langue préférée

Vous pouvez également changer la langue d’une visite ou d’un point d’intérêt (POI) spécifique en consultant les détails de la visite ou la page du POI en sélectionnant la langue audio disponible.

Pour une explication détaillée étape par étape, consultez la section Guide d’utilisation.

Pourquoi ne puis-je pas voir certaines visites dans ma ville ?

Q : Pourquoi ne puis-je pas voir certaines visites dans ma ville ?

R : VoxTour.ai se développe constamment, mais il est possible que des visites ne soient pas encore disponibles dans toutes les villes ou tous les lieux.

Si vous ne trouvez pas de visites dans votre ville, cela peut signifier :

  • Les visites n’ont pas encore été créées pour cette destination
  • Les visites sont en cours de préparation ou de mise à jour
  • La destination dispose actuellement d’un contenu limité

Si vous souhaitez une visite dans votre ville, vous pouvez nous contacter à contact@voxtour.ai, et nous créerons une visite pour vous gratuitement.

💡 Astuce :
Vous pouvez également explorer les villes voisines ou utiliser les outils d’exploration intelligente pour découvrir des points d’intérêt individuels.
