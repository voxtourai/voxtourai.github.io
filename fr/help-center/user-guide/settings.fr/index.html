Paramètres

VoxTour.ai vous permet de gérer facilement les identifiants de votre compte, vos préférences de langue et les paramètres de votre profil depuis le menu Paramètres.

Les paramètres sont disponibles sur mobile et sur ordinateur, et les fonctionnalités sont identiques sur toutes les plateformes.

Étape 1 : Ouvrir les paramètres

Sur mobile :

  1. Touchez votre avatar de profil en haut à droite.
  2. Touchez Paramètres.

Sur ordinateur :

  1. Cliquez sur votre avatar de profil en haut à droite.
  2. Sélectionnez Paramètres dans le menu déroulant.

Étape 2 : Ouvrir les paramètres généraux

Dans le menu Paramètres, touchez ou cliquez sur Général.

Cette section contient les identifiants de votre compte, vos préférences de langue et les options de gestion du compte.

Identifiants

La section Identifiants affiche votre adresse e-mail enregistrée et vous permet de mettre à jour vos informations de connexion.

Modifier l’adresse e-mail

  1. Touchez ou cliquez sur Modifier l’e-mail
  2. Saisissez :
    • Nouvelle adresse e-mail
    • Confirmer l’adresse e-mail
    • Mot de passe actuel
  3. Touchez ou cliquez sur Enregistrer

Si vous ne vous souvenez pas de votre mot de passe, déconnectez-vous et utilisez la réinitialisation du mot de passe.

Modifier le mot de passe

  1. Touchez ou cliquez sur Modifier le mot de passe
  2. Saisissez :
    • Mot de passe actuel
    • Nouveau mot de passe
    • Confirmer le nouveau mot de passe
  3. Touchez ou cliquez sur Enregistrer

Statut de connexion sociale

Cette section indique comment votre compte est connecté.

Exemple :

  • Compte local (e-mail et mot de passe)
  • Fournisseur de connexion sociale (si connecté)

Préférences de langue

Vous pouvez changer la langue de l’interface de l’application à tout moment.

Pour changer de langue :

  1. Touchez ou cliquez sur la langue actuelle
  2. Sélectionnez une nouvelle langue dans la liste
  3. L’application se met à jour automatiquement

Supprimer le compte

Vous pouvez supprimer définitivement votre compte VoxTour.ai depuis les Paramètres.

⚠️ La suppression de votre compte :

  • Supprime définitivement les données de votre compte
  • Ne peut pas être annulée
  • Peut supprimer les tours enregistrés et les préférences

Pour supprimer votre compte :

  1. Touchez ou cliquez sur Supprimer votre compte
  2. Confirmez votre décision lorsque cela vous est demandé

Sections de paramètres supplémentaires

Selon votre compte et les fonctionnalités activées, vous pouvez également voir :

  • VoxLens – Paramètres de reconnaissance d’images et de narration par IA
  • VoxRoute – Préférences de tours et de navigation
  • VoxExplore – Paramètres de découverte et d’exploration

Conseils

  • Gardez votre e-mail à jour pour assurer la récupération d’accès.
  • Utilisez un mot de passe robuste pour la sécurité.
  • Définissez votre langue préférée avant de commencer des tours pour une meilleure expérience.
