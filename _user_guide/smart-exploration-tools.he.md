## כלי חקירה חכמים (למובייל בלבד)

VoxTour.ai כולל כלי חקירה חכמים למובייל המסייעים לך לגלות סיפורים סביבך באופן אוטומטי או לזהות אתרים באמצעות בינה מלאכותית.

כלים אלו זמינים מתפריט **Smart Tools** בתוך האפליקציה הניידת.

---

### גישה לכלי חקירה חכמים

1. פתח את אפליקציית VoxTour.ai בנייד
2. פתח את מסך המפה
3. לחץ על כפתור **Smart Tools**
4. בחר:
    - **Navigator**
    - **Frame**

![Access Smart Exploration Tools]({{ site.baseurl }}/assets/images/smart-tools-menu.webp)

---

## Navigator

Navigator מספק סיפור אוטומטי מבוסס מיקום בזמן שאתה הולך או חוקר.

### כיצד Navigator פועל

כאשר אתה מפעיל את Navigator:

- המערכת מזהה את **המיקום הנוכחי שלך**
- VoxTour.ai מחפש את **נקודות העניין הקרובות ביותר (POIs)** סביבך
- כאשר אתה זז וחוצה גבולות של POI:
    - האפליקציה מפעילה את הסיפור באופן אוטומטי
    - המערכת קובעת **מאיזה כיוון אתה מתקרב**
    - הסיפור מותאם בהתאם למיקום ולתנועה שלך

Navigator מאפשר לך לחקור ערים באופן טבעי ללא בחירה ידנית של תחנות.

### כיצד להשתמש ב-Navigator

1. פתח Smart Exploration Tools
2. לחץ על **Navigator**
3. אפשר גישה למיקום אם מתבקש
4. התחל ללכת — VoxTour.ai ידריך אותך באופן אוטומטי

![Navigator]({{ site.baseurl }}/assets/images/smart-tools-navigator.webp)

## Frame (זיהוי תמונה)

Frame משתמש בזיהוי תמונה מבוסס בינה מלאכותית כדי לזהות אתרים וליצור סיפורי שמע.

### כיצד Frame פועל

כאשר אתה משתמש ב-Frame:

- אתה מעלה או מצלם תמונה של אתר או אובייקט
- VoxTour.ai מנתח את התמונה באמצעות בינה מלאכותית
- המערכת מזהה את המיקום או האובייקט
- סיפור או הסבר קשור נוצרים באופן אוטומטי

Frame מאפשר גילוי ספונטני ללא חיפוש או גלישה בסיורים.

### כיצד להשתמש ב-Frame

1. פתח Smart Exploration Tools
2. לחץ על **Frame**
3. העלה או צלם תמונה
4. המתן בזמן שה-AI מעבד את הבקשה
5. הסיפור ייווצר לאחר השלמת הזיהוי

![Frame]({{ site.baseurl }}/assets/images/smart-tools-frame.webp)

## טיפים לתוצאות הטובות ביותר

- הפעל שירותי מיקום בעת שימוש ב-Navigator
- השתמש בתמונות ברורות ומוארות היטב בעת שימוש ב-Frame
- עמוד מול האתר לקבלת דיוק זיהוי טוב יותר
- אפשר מספר רגעים לעיבוד AI בעת שימוש ב-Frame
