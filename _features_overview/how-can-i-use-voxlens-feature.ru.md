**В:** Как работает режим VoxRoute?

**О:** Функция **VoxLens** дает углубленные сведения о конкретных достопримечательностях. Просто наведите устройство на интересующий объект, и VoxLens предоставит подробный рассказ, помогая лучше понять место. Эта функция подходит тем, кто хочет получить более детальную информацию.

Как использовать VoxLens:

1. **Найдите направление**  
Воспользуйтесь поисковой строкой, чтобы найти место, которое хотите исследовать. По мере ввода появятся подсказки — выберите нужное направление, чтобы открыть его на карте.
    
    ![Поиск направления]({{ site.baseurl }}/assets/images/voxlens-landing-page-search-bar.webp)

2. **Найдите кнопку камеры на карте**  
После выбора направления вы увидите интерактивную карту с точками интереса. Найдите значок камеры, чтобы активировать VoxLens.

    ![Поиск направления]({{ site.baseurl }}/assets/images/voxlens-tour-overview-page.webp)

3. **Загрузите изображение**  
Выберите изображение из галереи устройства, на котором есть нужный объект. Приложение проанализирует выбранное изображение.
   
    ![Поиск направления]({{ site.baseurl }}/assets/images/voxlens-frame-load-image.webp)

4. **Распознавание изображения**  
VoxLens анализирует загруженное изображение и определяет объект. Если распознавание прошло успешно, вы можете добавить запрос, чтобы уточнить контекст или задать вопрос о месте.

    ![Поиск направления]({{ site.baseurl }}/assets/images/voxlens-frame-image-detection.webp)

5. **Страница аудиогида**  
Затем приложение формирует аудиогид с учетом настроек вашего профиля, например предпочтений по озвучке и интересам.

    ![Поиск направления]({{ site.baseurl }}/assets/images/voxlens-frame-page.webp)
