**P:** ¿Cómo puedo usar la función VoxLens?

**R:** La función **VoxLens** ofrece información detallada sobre lugares o sitios específicos. Solo apunta tu dispositivo a un punto de interés y VoxLens proporcionará narrativas detalladas, enriqueciendo tu comprensión del lugar. Esta función es ideal para quienes desean información más profunda sobre determinadas atracciones.

Para usar VoxLens:

1. **Busca tu destino**  
Usa la barra de búsqueda para encontrar el lugar que quieres explorar. A medida que escribes, aparecerán sugerencias. Selecciona el destino deseado para verlo en el mapa.
    
    ![Buscar un destino]({{ site.baseurl }}/assets/images/voxlens-landing-page-search-bar.webp)

2. **Encuentra el botón de la cámara en el mapa**  
Una vez seleccionado el destino, verás un mapa interactivo con varios puntos de interés. Busca el icono de la cámara para activar VoxLens.

    ![Buscar un destino]({{ site.baseurl }}/assets/images/voxlens-tour-overview-page.webp)

3. **Cargar imagen**  
Selecciona una imagen de la galería o biblioteca de fotos de tu dispositivo que muestre el lugar deseado. La app analizará la imagen seleccionada en busca de detalles relevantes.
   
    ![Buscar un destino]({{ site.baseurl }}/assets/images/voxlens-frame-load-image.webp)

4. **Detectar imagen**  
La función VoxLens analiza la imagen cargada para detectar e identificar el sitio. Si se identifica correctamente, puedes añadir un prompt para personalizar la experiencia. Este prompt permite aportar contexto adicional o preguntas sobre el lugar.

    ![Buscar un destino]({{ site.baseurl }}/assets/images/voxlens-frame-image-detection.webp)

5. **Página de guía de audio**  
La app generará una guía de audio específica para ti usando la configuración de tu perfil, como preferencias de narración y áreas de interés.

    ![Buscar un destino]({{ site.baseurl }}/assets/images/voxlens-frame-page.webp)
