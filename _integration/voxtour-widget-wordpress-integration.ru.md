# Плагин виджета VoxTour для WordPress – Руководство по установке и использованию

Плагин VoxTour позволяет легко встраивать аудиогиды VoxTour на ваш сайт с помощью шорткода.

## 1. Установка плагина

1. Скачайте **плагин VoxTour Widget (.zip)**, предоставленный VoxTour.
2. В панели управления WordPress перейдите в:

**Plugins → Add New → Upload Plugin**

3. Загрузите `.zip` файл плагина.
4. Нажмите **Install Now**.
5. Нажмите **Activate Plugin**.

---

## 2. Введите ваш API-ключ VoxTour

После активации плагина:

1. Перейдите в:

**Settings → VoxTour**

2. Вставьте **API Key**, предоставленный VoxTour.
3. Нажмите **Save Changes**.

Этот ключ соединяет ваш сайт с сервисом VoxTour.

---

## 3. Добавление аудиогида VoxTour на страницу или в запись

Вы можете отобразить любой аудиогид VoxTour, используя следующий шорткод:

[voxtour tour="TOUR_ID" locale="en"]

### Пример

[voxtour tour="29b24f5b-5b55-42bd-9964-7ae51f1a877b" locale="en"]

Где:

| Parameter  | Описание                                                     |
| ---------- |--------------------------------------------------------------|
| **tour**   | Уникальный ID тура, предоставленный VoxTour                 |
| **locale** | Язык тура (например `en`, `ru` и т.д.). По умолчанию `en`   |

---

## 4. Несколько туров на одной странице

Вы можете вставить несколько виджетов VoxTour на одной странице или в записи, добавив несколько шорткодов:

[voxtour tour="TOUR_ID_1" locale="en"]

[voxtour tour="TOUR_ID_2" locale="en"]

[voxtour tour="TOUR_ID_3" locale="en"]

---

## 5. Полноэкранный режим

Виджет VoxTour поддерживает полноэкранное воспроизведение.  
Пользователи могут развернуть аудиогид на весь экран прямо из плеера.

---

## 6. Аналитика

Плагин автоматически отслеживает загрузку виджета для систем аналитики сайта (например Google Analytics или Google Tag Manager, если они установлены).

---

## Нужна помощь?

Если вам нужны ID туров или доступ к API, свяжитесь с командой VoxTour.
