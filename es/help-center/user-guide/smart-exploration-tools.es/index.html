Herramientas de Exploración Inteligente (Solo Móvil)

VoxTour.ai incluye herramientas inteligentes de exploración móvil que te ayudan a descubrir historias a tu alrededor automáticamente o identificar monumentos utilizando IA.

Estas herramientas están disponibles desde el menú Smart Tools dentro de la aplicación móvil.

Acceder a Herramientas de Exploración Inteligente

  1. Abre la aplicación móvil VoxTour.ai
  2. Abre la pantalla del mapa
  3. Toca el botón Smart Tools
  4. Selecciona:
    • Navigator
    • Frame

Navigator

Navigator proporciona narración automática basada en la ubicación mientras caminas o exploras.

Cómo funciona Navigator

Cuando activas Navigator:

  • El sistema detecta tu ubicación actual
  • VoxTour.ai busca los Puntos de Interés (POIs) más cercanos a tu alrededor
  • A medida que te mueves y cruzas los límites de los POI:
    • La aplicación inicia automáticamente la historia
    • El sistema determina desde qué dirección te estás acercando
    • La historia se adapta según tu ubicación y movimiento

Navigator te permite explorar ciudades de forma natural sin seleccionar paradas manualmente.

Cómo usar Navigator

  1. Abre Smart Exploration Tools
  2. Toca Navigator
  3. Permite el acceso a la ubicación si se solicita
  4. Comienza a caminar — VoxTour.ai te guiará automáticamente

Frame (Reconocimiento de Imágenes)

Frame utiliza reconocimiento de imágenes con IA para identificar monumentos y generar historias de audio.

Cómo funciona Frame

Cuando usas Frame:

  • Subes o capturas una imagen de un monumento u objeto
  • VoxTour.ai analiza la imagen utilizando IA
  • El sistema identifica la ubicación u objeto
  • Se genera automáticamente una historia o explicación relacionada

Frame permite el descubrimiento espontáneo sin buscar ni navegar tours.

Cómo usar Frame

  1. Abre Smart Exploration Tools
  2. Toca Frame
  3. Sube o captura una imagen
  4. Espera mientras la IA procesa la solicitud
  5. La historia se generará una vez que el reconocimiento esté completo

Consejos para Mejores Resultados

  • Activa los servicios de ubicación cuando uses Navigator
  • Usa fotos claras y bien iluminadas cuando uses Frame
  • Colócate frente al monumento para mejorar la precisión del reconocimiento
  • Permite unos momentos para el procesamiento de IA al usar Frame

⚠️ Las Herramientas de Exploración Inteligente están disponibles solo en dispositivos móviles.
